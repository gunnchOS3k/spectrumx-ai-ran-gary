that would feed:

- Detector **retraining** and **calibration**
- **Digital-twin replay** (Gary site IDs)
- **ARC-OTA** / lab runtime correlation

**Status:** Integration-ready **target**; no OTA captures are bundled in the public repo.

See `docs/DATA_LAKE_SCHEMA.md` and `src/edge_ran_gary/ota_data_interface.py`.
